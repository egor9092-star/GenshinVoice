# Genshin Voice

Инструмент для автоматической озвучки диалогов в Genshin Impact с помощью OCR и синтеза речи.
Работает без внедрения в игру, поддерживает голос Windows и нейросетевые голоса Silero TTS.
Проект некоммерческий, создан для удобства игроков.

## Возможности
- Читает текст диалогов с экрана (OCR) и озвучивает его в реальном времени.
- Поддержка двух голосовых движков:
  - **Windows Voice** – встроенный русский голос Windows (SAPI).
  - **Silero Voice** – нейросетевые голоса (aidar, baya, eugene, kseniya, xenia, random) на основе Silero TTS.
- Загрузка модели Silero из официального источника одним кликом.
- Калибровка области сканирования под любой экран.
- Сохранение настроек (область, выбранный голос) между запусками.
- Пауза / продолжение озвучки.
- Лог работы внутри окна.

## Системные требования
- Windows 10 / 11 с видеокартой NVIDIA (поддержка CUDA).
- Установленный русский голос для SAPI (обычно уже есть).
- Для использования Silero Voice дополнительно требуется файл модели `model.pt`, который программа может скачать автоматически.

## Установка
### Способ 1: Готовая сборка (релиз)
1. Скачайте последнюю версию из [раздела Releases](https://github.com/egor9092-star/genshinvoice/releases).
2. Распакуйте архив в удобное место.
3. Запустите `GenshinVoice.exe`.

### Способ 2: Запуск из исходного кода
1. Убедитесь, что у вас установлен Python 3.8–3.11.
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/egor9092-star/genshinvoice.git
   cd genshinvoice
3. Установите зависимости: 
	pip install -r requirements.txt
4.Установите PyTorch с поддержкой CUDA с официального сайта. 	
	https://pytorch.org/get-started/locally/
5.Запустите: python genshin_voice.py

Использование

    Запустите программу и игру Genshin Impact.

    Нажмите «Калибровать область»: после 3‑секундного отсчёта выделите мышью прямоугольник вокруг текста диалогов. Область сохранится автоматически.

    Выберите голос: Windows Voice или Silero Voice. При первом выборе Silero модель предложат скачать с официального сайта – нажмите «Да».

    Озвучка включится автоматически, как только в игре появится диалог.

    Кнопка Пауза приостанавливает озвучку, не теряя прогресс.

Как это работает

Программа делает скриншот экрана, распознаёт текст с помощью EasyOCR и сразу преобразует его в речь. Никакие файлы игры не изменяются, код не внедряется в память процесса.
