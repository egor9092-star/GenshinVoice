Genshin Voice

Инструмент для автоматической озвучки диалогов в Genshin Impact с помощью OCR и синтеза речи.
Работает без внедрения в игру, поддерживает голос Windows и нейросетевые голоса Silero TTS.
Проект некоммерческий, создан для удобства игроков.

Возможности
- Читает текст диалогов с экрана (OCR) и озвучивает его в реальном времени.
- Поддержка двух голосовых движков:
  - Windows Voice – встроенный русский голос Windows (SAPI).
  - Silero Voice – нейросетевые голоса (aidar, baya, eugene, kseniya, xenia, random) на основе Silero TTS.
- Загрузка модели Silero из официального источника одним кликом.
- Калибровка области сканирования под любой экран.
- Сохранение настроек (область, выбранный голос) между запусками.
- Пауза / продолжение озвучки.
- Лог работы внутри окна.
- Встроенная проверка активности окна Genshin Impact: если игра не в фокусе, озвучка приостанавливается.

Системные требования
- Windows 10 / 11 (работает на любом процессоре, отдельная видеокарта не требуется).
- Установленный русский голос для SAPI (обычно уже есть в системе).
- Для использования Silero Voice требуется файл модели model.pt, который программа может скачать автоматически при первом переключении на этот голос.
- Tesseract OCR:
  - В готовой сборке (релиз) Tesseract уже включён в архив, устанавливать дополнительно ничего не нужно.
  - При запуске из исходного кода потребуется самостоятельно разместить Tesseract в папке bin/tesseract/ (см. Способ 2).

Установка

Способ 1: Готовая сборка (релиз)
1. Скачайте последнюю версию из раздела Releases: https://github.com/egor9092-star/GenshinVoice/releases
2. Распакуйте архив в удобное место.
3. Убедитесь, что рядом с GenshinVoice.exe находится папка bin\tesseract с файлами tesseract.exe и tessdata\rus.traineddata — они уже есть в архиве.
4. Запустите GenshinVoice.exe.

Способ 2: Запуск из исходного кода
1. Убедитесь, что у вас установлен Python 3.8–3.11.
2. Клонируйте репозиторий:
   git clone https://github.com/egor9092-star/GenshinVoice.git
   cd GenshinVoice
3. Установите основные зависимости:
   pip install -r requirements.txt
4. Установите PyTorch версии 2.4.1 (строго для процессора):
   pip install torch==2.4.1 --index-url https://download.pytorch.org/whl/cpu
5. Поместите Tesseract OCR в папку bin\tesseract\ рядом с main.py:
   - Скачайте портативную версию Tesseract (архив с бинарниками) с https://github.com/UB-Mannheim/tesseract/wiki или другого надёжного источника.
   - Создайте структуру:
     bin\tesseract\tesseract.exe
     bin\tesseract\tessdata\rus.traineddata
   - При необходимости добавьте eng.traineddata для корректного распознавания английских символов.
6. Запустите программу:
   python main.py

Использование
1. Запустите программу и игру Genshin Impact.
2. Нажмите «Калибровать область» — после трёхсекундного отсчёта экран затемнится, выделите мышью прямоугольник вокруг текста диалогов. Область сохранится автоматически.
3. Выберите голос: Windows Voice или Silero Voice. При первом выборе Silero модель предложит скачать с официального сайта — нажмите «Да».
4. Озвучка включится автоматически, как только в игре появится диалог. В верхнем правом углу окна отображается статус игры: «Genshin Impact активен» (зелёный) или «GenshinImpact.exe не активен» (красный). В нижней строке состояния будет написано «Область – ОК» (зелёный), если область диалога задана, или «Область не захвачена, откалибруйте область» (красный) — если область ещё не настроена.
5. Кнопка «Пауза» приостанавливает озвучку, позволяя временно остановить чтение, не сбивая настройки.

Как это работает
Программа делает скриншот указанной области экрана, предобрабатывает изображение (оставляя только текст), распознаёт его с помощью Tesseract OCR и сразу преобразует в речь через выбранный голосовой движок. Никакие файлы игры не изменяются, код не внедряется в память процесса.

Расположение Tesseract
- При запуске программа автоматически ищет Tesseract в папке bin\tesseract\ (рядом с исполняемым файлом или main.py). В готовой сборке также проверяется вложенный каталог _internal\bin\tesseract\, куда помещает Tesseract PyInstaller.
- Если Tesseract не найден, в интерфейсе загорится красная надпись «Не найден» и появится окно с подсказкой.
- Нажмите кнопку «Указать Tesseract» и выберите вручную путь к tesseract.exe. Рядом с ним обязательно должна находиться папка tessdata с языковым файлом rus.traineddata.
- Выбранный путь сохраняется в config.json и используется при последующих запусках.

Возможные проблемы и их решение

1. «Tesseract не найден» (красная надпись)
   - Убедитесь, что в папке bin\tesseract\ есть tesseract.exe и папка tessdata с файлом rus.traineddata.
   - Или укажите путь вручную через кнопку «Указать Tesseract».

2. Ошибка "Error opening data file rus.traineddata"
   - Проверьте, что rus.traineddata действительно лежит в папке tessdata.
   - Папка tessdata должна находиться на одном уровне с tesseract.exe. Программа задаёт этот путь автоматически, но при ручном указании убедитесь, что структура правильная.

3. Не озвучиваются цифры (например, «Номер 2»)
   - Программа автоматически заменяет одиночные цифры словами. Если замена не сработала, проверьте лог — туда попадает уже подготовленный текст. Некоторые голоса Silero всё равно произносят цифры корректно.

4. Статус «GenshinImpact.exe не активен» не меняется, хотя игра запущена
   - Убедитесь, что окно игры действительно в фокусе (заголовок должен содержать "Genshin Impact").
   - Окно не должно быть свёрнуто или полностью перекрыто другими программами.
   - Антивирусы или оверлеи (Discord, MSI Afterburner и пр.) иногда мешают определению окна — временно отключите их для проверки.

5. Распознанный текст содержит мусор (скобки, лишние символы)
   - Попробуйте точнее откалибровать область, захватив только строку диалога без элементов интерфейса.
   - В крайнем случае можно вручную отредактировать словари исправлений ocr_fixes и yo_map в файле main.py.

6. Ошибка загрузки модели Silero
   - При скачивании model.pt убедитесь, что есть доступ в интернет.
   - Если загрузка не удалась, скачайте модель вручную с https://models.silero.ai/models/tts/ru/v4_ru.pt и положите в папку с программой под именем model.pt.

7. При запуске из исходного кода возникает ошибка OSError: ... c10.dll ...
   - Это связано с конфликтом DLL при импорте PyTorch. Решение: перед первым запуском удалите файл config.json (если он существует) и убедитесь, что используется версия PyTorch для CPU. При старте программа переключится на Windows Voice и будет работать.

8. Ошибка c10.dll при использовании Silero Voice
   Эта ошибка возникает с некоторыми версиями PyTorch.
   Решение: удалите текущую версию (`pip uninstall torch -y`) и установите строго 2.4.1 для CPU:
   pip install torch==2.4.1 --index-url https://download.pytorch.org/whl/cpu
